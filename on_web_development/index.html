What allows all this to work?
Three main languages are used to build up websites.
And these are: HTML, CSS and Javascript.

HTML is as bare-bones as it can get. This is a markup language that allows text, images, so on and so forth to be displayed in a web browser. However, it isn't very pretty, nor is it particularly interactive.

CSS

CSS is a stylesheet language which brings flair to your HTML webpage. It even allows for minor interactivity. However, it is simply the second part of a larger whole.
